Linux 归档，压缩，解压命令速查

发表于2018-11-06|编程基础Linux
|字数总计:327|阅读时长:1分钟|阅读量:

example.tar 类型文件

打包

-c 表示创建包 ，-f指定包的文件名

增加到包

-r 表示增加文件

替换包中文件

-u 表示更新文件

列出所有文件

-t 表示列出文件

解压文件

-x 表示解开

example.tar.gz / example.tgz 类型文件

压缩

-c 表示压缩文件，-z表示gzip文件格式，-f 指定包名

解压

-x 表示解压

example.tar.bz2 类型文件

压缩

-c 表示压缩文件， -j 表示bzip2文件格式，-f 指定包名

解压

-x 表示解压

example.tar.Z 类型文件

压缩

-c 表示压缩文件， -Z表示bzip2文件格式，-f 指定包名

解压

-x 表示解压

example.gz 类型文件

压缩

gzip 压缩

解压

gunzip 表示解压 , -d 表示解压

example.bz2 类型文件

压缩

bzip2 压缩

解压

bunzip2 表示解压，-d 表示解压

example.zip 类型文件

压缩

zip 表示压缩文件

解压

unzip 表示解压
